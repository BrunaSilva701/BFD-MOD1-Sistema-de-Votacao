# Sistema de votação (Inspirado nas votações em conclaves)
## Estrutura
- Declaração de variáveis
- Array (vetores)
- Uso de ````for````, ````if```` e ````while````
- Menu para cadastro dos candidatos

## Regras do sistema
- Recebe candidatos
- Zera os votos antes de cada nova votação
- Percorrer o array mostrando os 5 cardeais cadastrado(menu)
- Recebe os votos
- Verifica se algum cardedal recebeu 60% ou mais dos votos
- Reinicia a votação se ningúem alcançou 60% ou mais votos
